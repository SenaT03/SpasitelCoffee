Our Work

A Spasitel Coffee package comes with a small eco-friendly pod plus an instruction manual on how to set-up the app. The pod is circular and waterproof with no health risks to the user, made from strong silicone caulk material to be specific, chosen for its anti-chemical, anti-flammable material which also provides water-proofing for the device. It comes equipped with a Bluetooth module with a programmed Link Manager that controls the connection to the user’s specific app through a GPO-like enforcement. This feature is provided to demote clutter of connections when the users are searching for their pod to connect through the app. The pod has heat-sensors installed inside the device which is programmed to alert the app whenever the temperature drops to 50 C. It is powered by a double AAA battery or optional recharging through power socket by using a usb-c type charging.

The app is supported for both IOS and Android phones which can be easily downloaded through Google Play or the Appstore. The app itself will have a basic interface to reduce guesswork and clutter for the user to navigate easy throughout the process. A circular connect button in the middle of the app which automatically detects and connects the nearby pod once the user has written the unique code of the pod which is prompted when the user opens the app for the first time. After the connection is established, a thermometer will pop up, displaying the current temperature of the pod’s area (The user can adjust between celsius or fahrenheit through the settings). The default alarm for the alarm is set for 50 C however, the user has the option to manually customize the temperature of when the alarm should alert. The user can reconfigure the alarm with the given alarm samples or upload their own custom alarm.
